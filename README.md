# FizzBuzz
My implementation of FizzBuzz in python3. Developed using TDD [check the commits ;)]

## Run Application

```bash
python3 main.py
```
When prompted type the max_range.

```python
Input...
> 10
1
2
Fizz...
```

## Running Tests

Install pytest
```bash
pip3 install pytest
```

Run the tests from the root directory
```bash
cd FizzBuzz
pytest -vx # v = verbose, x = halt on test failure
```
